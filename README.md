# VINID Mobile 🚗

VINID Mobile is a cross-platform mobile application built with React Native, Expo, and TypeScript.

The application connects vehicle owners with trusted mechanics, spare parts dealers, towing services, and other automotive professionals through a secure and user-friendly platform.

## Vision

To become Nigeria's most trusted digital platform for connecting vehicle owners with verified automotive service providers.

## Mission

To simplify access to reliable vehicle services by creating a secure, transparent, and technology-driven ecosystem for customers and automotive professionals.

## Core Features

- Customer Registration & Login
- Mechanic Registration & Login
- Vehicle Breakdown Assistance
- Find Nearby Mechanics
- GPS Navigation to Workshops
- Emergency Towing Requests
- Spare Parts Marketplace
- Secure Payments
- Ratings & Reviews
- Service History
- Real-time Notifications

## Getting Started

Clone the repository:

```bash
git clone https://github.com/patrickjnr11/vinid-mobile.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
